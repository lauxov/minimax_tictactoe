# Tic-Tac-Toe Game

![Tic-Tac-Toe](https://github.com/lauxov/minimax_tictactoe/assets/110423050/9c03987b-863e-4e49-85ed-992d363148ce) ![MiniMax2](https://upload.wikimedia.org/wikipedia/commons/e/e1/Plminmax.gif)


This is a simple implementation of the Tic-Tac-Toe game using Python and Tkinter. The game allows you to play against a computer bot with the minimax algorithm.

## Features

- Play against the computer bot
- Switch sides with the bot (X or O)
- Sound effects for each move
- Option to start a new game

## Prerequisites

Make sure you have the following installed:

- Python 3
- Tkinter module

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/lauxov/minimax_tictactoe.git
```
2. Change to the project directory:

  ```bash
  cd tic-tac-toe
  ```
4. Run the game:

  ```bash
  python x.py
  ```
  
## How to Play
> Click on any empty cell to make a move.
> The game will alternate between the player (X) and the computer bot (O).
> The first player to get three marks in a row, column, or diagonal wins the game.
> If there are no empty cells left and no winner, the game ends in a draw.

## Customize
You can customize the game by modifying the following:

>Sound effects: Replace the sound files in the sounds directory with your own WAV files.
>Game appearance: Adjust the button styles, fonts, colors, etc., in the create_ui method of the TicTacToe class.
